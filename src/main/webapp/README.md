### Spring-Boot-React-Template

Creating a PWA reactapp

* In Terminal type **cd ~/git/spring-boot-react-crud/src/main --template cra-template-pwa**
* In Terminal type **npx create-react-app webapp --template cra-template-pwa**

##### Inside application directory, you can run several commands

* In Terminal type **npm start** and press Enter. Starts the development server.
* In Terminal type **npm run build** and press Enter. Bundles the app into static files for production.
* In Terminal type **npm test** and press Enter. Starts the test runner.
* In Terminal type **npm run eject** and press Enter. Removes this tool and copies build dependencies, configuration
  files and scripts into the app directory. If you do this, you canâ€™t go back!

### Build and run the application

* In Terminal type **cd ~/git/react-template** and press Enter
* In Terminal type **npm start** and press Enter

##### Running PWA

* In Terminal type **cd ~/git/react-template** and press Enter
* In Terminal type **npm run build** and press Enter
* In Terminal type **npm install --location=global server** and press Enter
* In Terminal type **serve -s build** and press Enter
